# Titanic Survival Prediction 

## About This Project
Built a machine learning model to predict whether a passenger 
would survive the Titanic disaster using real historical data.

## What I Found (Key Insights)
- Women survived 4x more than men (74% vs 19%)
- 1st class passengers survived 2.5x more than 3rd class (63% vs 24%)
- These 2 patterns became the most powerful features in the model

## Model Performance
- Algorithm: Decision Tree Classifier
- Accuracy: 77.62%
